# Orecce Web Tester

Simple web interface for testing the AI-post backend locally.

## Run
From repo root (recommended):
1. `npm run start:all`
2. Open `http://127.0.0.1:5173`

Or web-only:
1. Install:
   - `npm install`
2. Start:
   - `npm run dev`
3. Open:
   - `http://127.0.0.1:5173`

## Backend target
Default base URL:
- `http://127.0.0.1:5001/ai-post-dev/us-central1/api`

You can change the base URL in the app if your backend runs elsewhere.
